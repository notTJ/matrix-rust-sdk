Expand description

Data for a request to the /keys/upload API endpoint (specification).

Publishes end-to-end encryption keys for the device.

Fields

id: String

The request ID.

body: String

A JSON-encoded object of form:

{"device_keys": …, "one_time_keys": …, "fallback_keys": …}
